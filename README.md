# 🚀 Team Task Manager

A premium, full-stack project management platform designed for modern teams. This application provides a high-performance dashboard to track tasks, manage projects, and monitor team performance in real-time.

## ✨ Key Features

### 📊 Professional Dashboard
*   **Real-time Stats**: Track active projects, total tasks, and completion rates at a glance.
*   **Ongoing Projects**: Quick access to your most important active work.
*   **Visual Analytics**: High-end donut charts showing task status distribution (Completed, In Progress, Pending, Overdue).

### 🛠️ Core Functionality
*   **Project Management**: Create, edit, and archive projects with unique IDs and detailed descriptions.
*   **Task Tracking**: Assign tasks to team members with due dates and status cycling.
*   **Role-Based Access**: Admins can manage projects and members, while team members focus on task execution.
*   **Team Insights**: Detailed profile views showing project assignments and task history for every member.

### 🎨 Premium UI/UX
*   **Gender-Aware Avatars**: Automated profile picture selection based on team member names.
*   **Dynamic Toast Notifications**: Beautiful, non-intrusive alerts for all system actions.
*   **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
*   **Dark-Mode Elements**: High-contrast sidebar and visual elements for a premium feel.

---

## 💻 Tech Stack

*   **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+)
*   **Backend**: Node.js, Express.js
*   **Database**: SQLite3 (with automated seeding for production)
*   **Auth**: JWT (JSON Web Tokens) with Bcrypt password hashing
*   **Deployment**: Railway (CI/CD via GitHub)

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18.0.0 or higher)
*   npm

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/Jaiswalradha335/Team-task-manager.git
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:3000`
