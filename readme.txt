TEAM TASK MANAGER - PROFESSIONAL PROJECT MANAGEMENT

PROJECT OVERVIEW:
A premium, full-stack project management platform designed for 
modern teams to track tasks, manage projects, and monitor 
performance in real-time.

LIVE PROJECT URL:
https://team-task-manager-production.up.railway.app/

KEY FEATURES:
- Dynamic Dashboard with Real-time Statistics
- Ongoing Project Tracking & Management
- Visual Analytics (Task Status Distribution)
- Role-Based Access Control (Admin/Member)
- Gender-Aware Profile Avatars
- Responsive & Premium User Interface

TECH STACK:
- Frontend: HTML5, Vanilla CSS3, JavaScript (ES6+)
- Backend: Node.js, Express.js
- Database: SQLite3
- Authentication: JWT & Bcrypt
